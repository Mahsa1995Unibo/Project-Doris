SPARQL Queries

Querying the Knowledge Graph

Four queries that answer our research questions. All six mandatory keywords are present across these queries.

Mandatory keywords — all used
DISTINCT (Query A, B, C)  ·  OPTIONAL (Query B)  ·  UNION (Query C)  ·  FILTER + REGEX (all queries)  ·  LIMIT (all queries)  ·  ORDER BY (Query A, B, D)

Query A — Who mentions "Jews" and in what role?

Research question answered: Which semantic roles are assigned to Jewish entities across the corpus? This is our starting point — a broad scan across all three languages.

Query A · DISTINCT · FILTER · REGEX · ORDER BY · LIMIT
# Find all text chunks mentioning Jewish people, with their roles and language
SELECT DISTINCT ?text ?role ?lang
WHERE {
  ?dec :isDecodingOf ?enc ;
       :decodedAs ?role .
  ?enc :encodes ?chunk .
  ?chunk :hasRealization ?r .
  ?r rdfs:label ?text .
  ?dec :inDocument ?doc .
  ?doc :hasLanguage ?lang .
  FILTER (REGEX(LCASE(?text),
    "jews|jewish|juden|ebrei|flüchtlinge|refugees", "i"))
}
ORDER BY ?lang ?role
LIMIT 200

Interpretation
English sources consistently assign the Victim role. German sources assign Assailant or Source roles — framing Jewish people as something to be removed. Italian sources lean toward the passive Theme role.

Query B — Count roles per language

Research question answered: Is the Victim role assigned more often in English than in German? This query gives us numbers to compare.

Query B · OPTIONAL · GROUP BY · ORDER BY · LIMIT
# Count how many times each role appears per language
# OPTIONAL: some articles may not have a date — we include them anyway
SELECT ?lang ?role (COUNT(*) AS ?total)
WHERE {
  ?dec :isDecodingOf ?enc ;
       :decodedAs ?role .
  ?enc :encodes ?chunk .
  ?chunk :hasRealization ?r .
  ?r rdfs:label ?text .
  ?dec :inDocument ?doc .
  ?doc :hasLanguage ?lang .
  OPTIONAL { ?doc :hasDate ?date }
  FILTER (REGEX(LCASE(?text),
    "jews|jewish|juden|ebrei", "i"))
}
GROUP BY ?lang ?role
ORDER BY ?lang DESC(?total)
LIMIT 100

Query C — Find Victims AND Assailants in one query

Research question answered: When the same entity appears as both Victim in one language and Assailant in another — what does the text actually say?

Query C · UNION · DISTINCT · FILTER · LIMIT
# UNION combines two patterns into one result set
# This finds Jewish entities in EITHER the Victim OR Assailant role
SELECT DISTINCT ?text ?role ?lang
WHERE {
  {
    ?dec :decodedAs
      <framester:fe/Victim.attack> .
  }
  UNION
  {
    ?dec :decodedAs
      <framester:fe/Assailant.attack> .
  }
  ?dec :isDecodingOf ?enc .
  ?enc :encodes ?chunk .
  ?chunk :hasRealization ?r .
  ?r rdfs:label ?text .
  ?dec :inDocument ?doc .
  ?doc :hasLanguage ?lang .
  FILTER (REGEX(LCASE(?text),
    "jews|jewish|juden|ebrei", "i"))
}
LIMIT 150

Query D — Before vs After Kristallnacht

Research question answered: Does the framing of Jewish people shift after November 9, 1938?

Query D · BIND · FILTER · GROUP BY · ORDER BY · LIMIT
# Split results into "before" and "after" Kristallnacht
SELECT ?role ?period (COUNT(*) AS ?n)
WHERE {
  ?dec :isDecodingOf ?enc ;
       :decodedAs ?role .
  ?enc :encodes ?chunk .
  ?chunk :hasRealization ?r .
  ?r rdfs:label ?text .
  ?dec :inDocument ?doc .
  ?doc :hasDate ?date .
  FILTER (REGEX(LCASE(?text),
    "jews|jewish|juden|ebrei", "i"))
  BIND (IF(?date < "1938-11-09"^^xsd:date,
    "before Kristallnacht",
    "after Kristallnacht") AS ?period)
}
GROUP BY ?role ?period
ORDER BY ?period DESC(?n)
LIMIT 100

Key finding
The Victim role assignment increases sharply after Kristallnacht — at least in English-language sources. This confirms that the scale of the violence shifted how newspapers framed Jewish people: from a refugee crisis to a clear victim narrative.
